## 概述

Xkit一个效率小工具，软件插件化设计。使用不同的插件达到不同的功能目的，也可以开发属于自己的插件。

## 运行

1. 按照[Tauri官方文档](https://v2.tauri.app/start/prerequisites/)安装运行所需要依赖
2. 使用`pnpm i`安装依赖
3. 使用`pnpm dev`运行项目
